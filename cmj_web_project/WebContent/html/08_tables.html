Tables 3행(tr) 4열(td)

tr에는 border가 적용 안됨, td에는 적용됨

border에는 두께(1px), 스타일(solid 직선,점선), 색상 지정할 수 있음
border-collapse : collapse; border 사이 공백 제거
padding : 글자 시작, 끝 양 옆에 공백
padding-top 위에만 패딩 생성, right, bottom, left
padding : 10(t) 12(r) 10(b) 12(l);
padding : 10px 20px => top, bottom : 10px right,left : 20px
colspan = "2" 2열을 합침, rowspan = 행을 합침
tr:nth-child(even) { background-color: #D6EEEE; =tr의 짝수에만 색을 넣어줌 }
