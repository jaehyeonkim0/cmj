Block & inLine Tags

h*, p, div 태그들이 대표적인 Block level(태그)

p태그는 css를 적용해보면 페이지의 한 줄을 다 차지한다 => 이렇게 한 줄을 다 차지하는 태그들을 block level

div 태그 역시 한 줄을 다 차지 = Block Level

H3 태그 = Block Level = 들어가는 데이터의 크기는 작을 수 있지만 "한 줄을 모두 쓴다"

hr = 라인 하나 긋는 태그

Span 태그 = 한 줄을 다 차지 하지 않고, 데이터가 있는 만큼만 영역이 만들어짐 = inline tag

Span 태그 - img 파일 "옆"에 이미지 파일 출력됨
